## Project Overview

This project was developed as part of **Task 2 вЂ“ Interactive UI & Frontend Development** under the ApexPlanet Software Internship Program.

The objective of this task was to create a responsive, user-friendly, and interactive web interface using HTML, CSS, JavaScript, and Bootstrap.

## Features

* Responsive Home Page
* User Registration Page
* User Login Page
* Email Validation using Regular Expressions (Regex)
* Password Validation using JavaScript
* Confirm Password Matching
* Show/Hide Password Functionality
* Responsive Design using Bootstrap 5
* Custom Styling and Hover Effects
* Font Awesome Icons Integration
* Mobile-Friendly User Interface

## Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* Bootstrap 5
* Font Awesome

## Pages

### Home Page

The landing page of the website that provides navigation to Login and Registration pages.

### Registration Page

Allows new users to register by entering their details. The form includes:

* Email Validation using Regex
* Password Validation
* Confirm Password Matching

### Login Page

Allows existing users to log in using their credentials. The page includes:

* Email Validation using Regex
* Password Validation
* Show/Hide Password Feature

## Validation Features

### Email Validation

Implemented using JavaScript Regular Expressions (Regex) to ensure valid email formats.

### Password Validation

Checks password requirements before form submission.

### Confirm Password Validation

Ensures both password fields match during registration.

## Responsive Design

Bootstrap 5 was used to create a fully responsive layout that works across:

* Desktop Devices
* Tablets
* Mobile Phones

## Note

This project is developed using frontend technologies only (HTML, CSS, JavaScript, and Bootstrap). No backend or database has been used; therefore, user registration data is not stored permanently.
